# Automated Client Onboarding System (Clio + Zapier)

## Overview

This project demonstrates a client onboarding automation system designed for law firms using Clio and Zapier.

When a new client is added in Clio, the system automatically sends a personalized welcome email. This ensures fast response time, consistent communication, and a more professional onboarding experience.

## Workflow

Clio (New Contact)
